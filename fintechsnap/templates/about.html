FinMentor is a data-driven personal finance analysis system that provides a financial health score, risk insights, and actionable recommendations using machine learning techniques — without requiring bank integrations or complex tracking.

1. Enter Financial Details

Provide income, expenses, savings, and EMI details for analysis.

2. Data-Driven Analysis

Financial health is evaluated using an ensemble of machine learning models.

3. Insights & Recommendations

Receive a financial health score, risk assessment, and actionable steps for improvement.

CORE FEATURES

Machine Learning-Based Analysis

Uses an ensemble of SVM, CRF, and HMM models for reliable financial assessment.

Financial Health Scoring

Generates a structured score along with persona classification (Stable, Moderate, Stressed).

Risk & Affordability Analysis

Evaluates EMI load, savings rate, and income ratios to determine financial risk levels.

Interactive Dashboard

Displays key financial insights and metrics in a clear and user-friendly format.

Privacy & Data

FinMentor is developed as an academic project. The system does not require bank credentials and does not store sensitive financial data. User inputs are processed within the session environment. Future extensions involving persistent storage would include explicit user consent and a defined privacy policy.

FAQ

Do I need to connect my bank?
No — the system is designed to be simple and privacy-focused. No bank integration is required.

Is the score accurate?
The score is a data-driven estimate based on financial indicators and machine learning models. It is intended as a decision-support tool and can be further refined with additional data.
